### Item 1 â€” iOS sticky send bar on /deep-chat
`journal-write-page.tsx` (the unified compose view) was restructured from a scrolling page + `position:fixed` bottom composer (which slid under the iOS keyboard on scroll) into a viewport-PINNED flex column:
- Outer shell: `position:fixed; inset-x-0; top:var(--vv-top); height:var(--app-vh)` + `flex flex-col overflow-hidden`.
- Vars from `use-visual-viewport.ts`: `--app-vh = visualViewport.height`, `--vv-top = visualViewport.offsetTop`, updated on vv `resize`+`scroll`.
- Middle: `flex-1 overflow-y-auto overscroll-contain` (ONLY the thread scrolls).
- Composer: now the LAST `flex-shrink-0` child (no longer independently `position:fixed`), with `pb-[max(0.5rem,env(safe-area-inset-bottom))]`.
- `/deep-chat` is not in `BOTTOM_NAV_PATHS`, so the full-viewport shell has no nav chrome to clash with.
